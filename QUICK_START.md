# Vitallic - Quick Start Guide

## What You Have

A fully functional Dutch AI voice assistant called **Vitallic** with:
- Real-time voice conversations in Dutch
- Two voice personalities (Friendly & Professional)
- Web dashboard for monitoring calls
- Complete conversation history and transcripts
- Supabase database integration

## Getting Started

### 1. Open the Application
The application is running and ready to use at your configured URL.

### 2. Test the Voice Assistant
1. Click the "Assistent" tab
2. Select a voice profile (Friendly & Casual or Professional & Business-Friendly)
3. Click the blue microphone button
4. **Grant microphone permissions** when prompted
5. Speak in Dutch (try: "Hallo, hoe gaat het?")
6. Listen to Vitallic's response
7. Continue the conversation naturally
8. Click the red microphone to stop

### 3. View the Dashboard
1. Click the "Dashboard" tab
2. See call statistics (total calls, duration, etc.)
3. View conversation history
4. Click any call to see its full transcript

## Browser Requirements

**Use Google Chrome or Microsoft Edge for best results**

## Demo Script

Try this conversation:
