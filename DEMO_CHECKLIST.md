# Vitallic Demo Checklist

## Pre-Demo Setup (5 minutes)

### Browser & System
- [ ] Use Google Chrome or Microsoft Edge
- [ ] Check internet connection is stable
- [ ] Ensure microphone is working and not muted
- [ ] Close unnecessary applications
- [ ] Disable notifications

### Application
- [ ] Application is running and responsive
- [ ] Tabs are clickable (Assistent / Dashboard)
- [ ] Voice profiles are loading
- [ ] No console errors visible

## Demo Script (15-20 minutes)

### Part 1: Introduction (2 minutes)
1. **Show the interface**: "This is Vitallic, a Dutch AI voice assistant"
2. **Highlight key areas**: Voice profile selector, microphone button, dashboard
3. **Explain functionality**: Real-time conversations, call tracking, transcripts

### Part 2: Friendly Voice Profile Demo (5 minutes)
1. **Select Profile**: Click "Friendly & Casual" voice profile
2. **Start Conversation**: Click blue microphone button
3. **Grant Permissions**: Allow microphone access when prompted
4. **First Interaction**:
   - Say: "Hallo Vitallic"
   - Vitallic: "Hallo! Leuk je te horen. Hoe gaat het met je?"
5. **Second Interaction**:
   - Say: "Goed hoor, en met jou?"
   - Vitallic: "Met mij gaat het uitstekend, dank je! En met jou?"
6. **Third Interaction**:
   - Say: "Wat kun je allemaal doen?"
   - Vitallic: "Ik kan je helpen met verschillende taken, gesprekken voeren, en vragen beantwoorden..."
7. **Thank You**:
   - Say: "Bedankt!"
   - Vitallic: "Graag gedaan! Altijd fijn om te helpen!"
8. **Stop**: Click red microphone button

### Part 3: Professional Voice Profile Demo (3 minutes)
1. **Switch Profile**: Select "Professional & Business-Friendly"
2. **Start New Conversation**: Click blue microphone
3. **Professional Interaction**:
   - Say: "Goedendag"
   - Vitallic: "Goedendag. Waarmee kan ik u helpen?"
4. **Show Tone Difference**: Point out the more formal language and tone
5. **Stop**: Click red microphone

### Part 4: Dashboard Demonstration (5 minutes)
1. **Switch to Dashboard Tab**: Click "Dashboard"
2. **Show Statistics**:
   - Point to "Totaal Gesprekken" (should show 2)
   - Point to "Totale Duur" (combined duration)
   - Point to "Actieve Gesprekken" (0, since we stopped them)
3. **Call History**:
   - Show the list of conversations
   - Highlight timestamps and durations
   - Show status indicators (Voltooid = Completed)
4. **View Transcript**:
   - Click on first call to view full transcript
   - Show blue messages (user) and gray messages (assistant)
   - Point out timestamps and confidence scores
   - Highlight that every message is tracked

### Part 5: Key Features Highlighting (3 minutes)
While viewing dashboard, explain:
1. **Real-time Recording**: Every word is captured instantly
2. **Conversation Tracking**: Complete history available
3. **Multiple Profiles**: Easy to switch between personalities
4. **Responsive Design**: Works on mobile and desktop
5. **Secure Storage**: All data in encrypted database

## Troubleshooting During Demo

### Microphone Not Working
- Solution: Refresh page, check browser permissions, try different browser
- Fallback: Show dashboard from previous test run

### Speech Not Recognized
- Solution: Speak more clearly, reduce background noise
- Fallback: Manual testing later, show video demo

### Dashboard Not Loading
- Solution: Refresh page, check Supabase connection
- Fallback: Show data directly from browser console

### Performance Issues
- Solution: Close other tabs, reduce browser extensions
- Fallback: Come back in 5 minutes after clearing cache
